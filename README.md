## 📌 Project Overview

The objective of this project was to convert a provided Figma design into a fully responsive static website while maintaining:

- ✅ Pixel-perfect UI accuracy
- ✅ Responsive layouts across all device sizes
- ✅ Clean, maintainable code structure
- ✅ Semantic HTML5 markup
- ✅ Modern frontend best practices

---

## ✨ Features

- 🎨 Pixel-perfect Figma-to-code implementation
- 📱 Fully responsive design (mobile-first approach)
- 🧭 Responsive navbar with mobile hamburger menu
- 🔤 Adaptive typography system
- 🃏 Responsive cards and content sections
- ✨ Hover effects and interactive button states
- 🏗️ Semantic HTML5 structure
- 🧹 Clean and maintainable CSS architecture

## 🚀 Getting Started

### Prerequisites

No build tools or dependencies required — just a browser.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Dashwanth15/infotech-frontend-assignment.git

# Navigate into the project directory
cd infotech-frontend-assignment

# Open index.html in your browser
open index.html
```

Or use a local dev server like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code for the best experience.
